# my personal config files

I've switched to Debian 12 with sway

## Installation
`````sh
# clone this repo
git clone https://github.com/vazw/dotfiles
cd dotfiles
# Install sway and it's dependencies
./install.sh

# Install Nerd Fonts
./nerdfonts.sh

# if need neovim
./neovim.sh
`````
